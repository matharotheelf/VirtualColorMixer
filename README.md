## Getting Started

### Prerequisites

The hardware needed for this project are the following:

1. Arduino Microcontroller
2. Computer with Arduino installed
3. 1 RGB LED
4. 5 Button Switches
5. 5 10K Ohm resistors
6, PIR Motion Sensor

### Installation

1. Install the [Arduino IDE](https://docs.arduino.cc/software/ide-v1)

2. Clone this repo

3. Open the file `.ino` in the IDE.

4. Upload the code to a connected Arduino.

### Hardware Setup

The circuit Diagram for the project is as follows:

![ColourMVPCircuit](https://github.com/matharotheelf/VirtualColorMixer/assets/44533664/7d4d5f1b-9cec-42c4-85a7-f52e09d8d47a)

## Usage

To use:

1. Select your colour by pressing a button
2. Trigger the motion sensor by moving your hand to mix your colour and see the colour of the light change.
